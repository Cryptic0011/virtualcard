# Virtual Business Card

This is a modern, mobile-optimized virtual business card built with React and Tailwind CSS.

## Features

- **vCard Generation**: "Save Contact" button adds your details to the user's phone contacts.
- **Responsive Design**: Looks great on mobile and desktop.
- **Modern Aesthetic**: Glassmorphism, gradients, and smooth animations.
- **SEO Optimized**: Meta tags included for better sharing.

## Project Structure

- `src/App.jsx`: Main component containing the layout and logic.
- `public/`: Contains assets like `pfp.jpg` and `logo.jpg`.
- `tailwind.config.js` / `postcss.config.js`: Styling configuration.

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd app
    ```
2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser at `http://localhost:5173`.

## Customization

- **Update Info**: Edit the `contactInfo` object in `src/App.jsx`.
- **Change Images**: Replace `pfp.jpg` and `logo.jpg` in the `public` folder.
- **Colors**: Modify Tailwind classes in `src/App.jsx` to match your brand colors.
